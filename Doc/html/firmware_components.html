Firmware components

The Smart Remote application is built on the following components.

Modules

Input modules

  • m_acc Low power accelerometer module.
  • m_audio Audio module that handles voice input.
  • m_audio_frame Audio frame management module.
  • m_batt_meas Battery monitoring module.
  • m_gyro Gyroscope module that handles motion processing.
  • m_keyboard Keyboard module for handling keypad events.
  • m_touchpad Touchpad module that deals with touch sensor data.

Output modules

  • m_buzzer Buzzer module.
  • m_coms Module that handles high-level communication with host.
    • m_coms_ble Submodule that implements low-level communication procedures.
    • m_coms_ble_addr Submodule that serves as flash address storage utility.
    • m_coms_ble_adv BLE advertising submodule.
    • m_coms_ble_atvv Voice over BLE for Android implementation submodule.
    • m_coms_ble_dfu BLE Device Firmware Upgrade submodule.
    • m_coms_ble_hid BLE HID over GATT submodule.
    • m_coms_ble_lesc Low Energy Secure Connection submodule.
    • m_coms_ble_atvv Voice over BLE for Android TV implementation (experimental feature).
  • m_ir_tx Infrared transmitter module.
  • m_leds LED module that handles LED signaling.
  • m_nfc NFC module that provides support for the tap-to-pair feature.

System state and protocol modules

  • m_init Initialization module that calls initialization functions for all enabled modules.
  • m_protocol_hid HID protocol module.
  • m_protocol_ir IR protocol module.
  • m_system_state System state module.

Drivers

  • drv_acc Low power accelerometer driver.
  • drv_audio Audio driver.
  • drv_audio_codec Audio codec driver.
  • drv_audio_dsp Audio DSP virtual driver.
  • drv_board Board driver.
  • drv_gyro Gyroscope driver.
  • drv_ir Driver for the infrared protocol.
  • drv_keyboard Keyboard driver.
  • drv_leds LED driver.
  • drv_touchpad Touchpad driver.

Debug utilities

  • app_debug_pin Pin-based debug utilities.

Other components

  • app_error Common application error handler.
  • app_isched Instantiable scheduler.
  • app_scheduler A custom implementation of the app_scheduler that replaces the SDK app_scheduler.
  • event_bus Event bus module.
  • resources Shared hardware resource configuration.
  • twi_common Common TWI procedures.
